# Tools Guide

This document provides a comprehensive reference for the tools integrated into the Harbinger platform, categorized by their primary function.

## Reconnaissance

| Tool | Description |
|---|---|
| `subfinder` | A passive subdomain discovery tool that enumerates subdomains from various sources. |
| `dnsx` | A fast and multi-purpose DNS toolkit that allows running multiple DNS queries. |
| `httpx` | A fast and multi-purpose HTTP toolkit that allows running multiple probers using retryablehttp library. |
| `naabu` | A fast port scanner written in Go with a focus on reliability and simplicity. |
| `nmap` | A powerful network scanner for network discovery and security auditing. |
| `masscan` | An incredibly fast TCP port scanner. |

## Web Application Testing

| Tool | Description |
|---|---|
| `sqlmap` | An open-source penetration testing tool that automates the process of detecting and exploiting SQL injection flaws. |
| `dalfox` | A powerful open-source XSS scanner and parameter analyzer. |
| `xsstrike` | A comprehensive XSS scanner that can detect and exploit XSS vulnerabilities. |

## OSINT

| Tool | Description |
|---|---|
| `theHarvester` | A tool for gathering open-source intelligence (OSINT) from various public sources. |

## Reporting

| Tool | Description |
|---|---|
| `CVSS Calculator` | A tool to calculate Common Vulnerability Scoring System scores. |
